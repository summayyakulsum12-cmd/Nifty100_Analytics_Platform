# Nifty100 Analytics Platform

Nifty100 Analytics Platform is a full-stack financial analytics system designed to analyze and visualize the financial performance of Nifty 100 companies.

The project integrates ETL pipelines, a PostgreSQL data warehouse, machine learning models, and an interactive Django web application to provide insights into company fundamentals and financial health.

## Key Features

* Financial dashboards for Nifty 100 companies
* Revenue, Profit, EPS, Dividend and Cash Flow visualization
* Company Health Score using Machine Learning
* Stock Screener based on financial metrics
* Company Comparison Dashboard
* Similar Company Recommendation Engine
* Sales Forecasting and Trend Analysis
* PostgreSQL Data Warehouse for structured financial storage
* REST APIs for financial data access
* Responsive web-based user interface
